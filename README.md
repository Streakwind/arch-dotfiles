# arch-dotfiles
my dotfiles for hyprland 

dependencies
```
hyprland zsh fastfetch oh-my-posh-bin git base-devel kitty catppuccin-gtk-theme-mocha nwg-look
```

install w/ gnu stow
```
sudo pacman -S stow
stow -d ~/wherever you cloned this repo -t ~/ dotfiles
```

.poshthemes from https://github.com/JanDeDobbeleer/oh-my-posh/tree/main/themes
